Quien soy?

Soy una persona responsable, creativa, con iniciativa y puntualidad, me caracterizo por poseer una capacidad de liderazgo ético y facilidad de aprendizaje así como una amplia disposición para integrarme en grupos de trabajo asumiendo responsabilidades individuales y de equipo.

He logrado adquirir habilidades comunicativas manejando puntos reflexivos y críticos; tengo grandes capacidades para adaptarme al cambio y buen manejo de relaciones interpersonales.

Poseo excelente presentación personal y una amplia visión estratégica en el manejo de herramientas informáticas, desarrollo web (back-end y front-end) y aplicativos moviles, así como también gran capacidad de diseño, análisis y programación.

De la misma manera deseo adquirir experiencia en trabajo que demande desarrollar funciones que involucren creatividad, innovación, liderazgo, investigación y trabajo en equipo a fin de enriquecerme en el ámbito personal y profesional.

Proyectos

Imagen propia del autor (prototipo real)

LegionApp es un aplicativo web que tiene como fin dar control a el registro de deportista de un club, escuela de formacion o liga deportiva. Tambien se encarga de registrar los datos relevantes (medidas antropometricas, rendimiento fisico, etc.) en una BD NoSql con el proposito de analisarlos y poder optener informacion de ellos.

Este proyecto esta desarrollado en el lado del back-end con Node.js, express, cors, validator, JsonWebToken, bcryptjs, moment y mongoDB; por el lado del front-end esta desarrollado con React, moment, redux.

Aun esta en desarrollo por lo cual no hay un link directo al proyecto, aun asi espero poder enseñarlo lo antes posible.

Habilidades

Soy estudiante de 8vo semestre de ingenieria de software, desde hace un año me introduje en el mundo del desarrollo web ya que me resulta muy apacionante.

Mi ruta de aprendizaje fue JavaScript con node.js y react, a base de cursos y ser autodidacta. Por el tema de la universidad yo tenia conocimientos previos sobre Python y Java, por lo cual fue un poco mas facil profundizar en esos lenguajes.

En mi experiencia con proyectos personales he usado mayormente JavaScript con sus distintos framworks y python con la intencion de analisis de datos e interactuando mediante una api con la parte del cliente (React)

JavaScript

El lenguaje que mas manejo es JavaScript, llevo poco mas de un año aprendiendolo y usandolo en mis proyectos personales y universitarios. Junto a el manejo el framework de node.js en el cual uso express, JsonWebToken, bcrypt, etc. Tambien manejo React para el lado del cliente lo cual hace un trabajo maravilloso al dar una UX muy buena.

React

React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.

Llevo un año manejando esta libreria y debo decir que me gusta mucho el poder desarrollar interfaces agradables y veloces en poco tiempo. Sus hooks son muy utilies y ayudan a tener un gran control de lo que quiero desarrollar

Node.js

Node.js es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor basado en el lenguaje de programación JavaScript, asíncrono, con E/S de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google.

Gracias a node.js me intruduje en el mundo de JavaScript, y comence mi ruta de aprendizaje junto a express y otras librerias para el back-end.

Python

Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código.​ Se trata de un lenguaje de programación multiparadigma, ya que soporta parcialmente la orientación a objetos, programación imperativa y, en menor medida, programación funcional.

Con python me enfoco en el tratamiento de big data e inteligencia artificial aunque es en menor medida a comparacion de JavaScript.
